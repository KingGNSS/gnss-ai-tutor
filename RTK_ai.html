非差观测方程 (Undifferenced)：
这是最原始的GNSS原始载波相位观测方程（对应您图中的公式结构）。每个方程都饱受接收机钟差、卫星钟差以及电离层/对流层延迟的影响。
LA1 = ρA1 + c·dtA - c·dt1 - IA1 + TA1 + λNA1
LB1 = ρB1 + c·dtB - c·dt1 - IB1 + TB1 + λNB1
LA2 = ρA2 + c·dtA - c·dt2 - IA2 + TA2 + λNA2
LB2 = ρB2 + c·dtB - c·dt2 - IB2 + TB2 + λNB2
站间单差 (Between-Receiver Single Difference)：
两台接收机对同一颗卫星作差。可完全消除卫星钟差，大气的消除效果取决于下方的基线长短设置。
ΔLAB1 = ΔρAB1 + c·dtAB + λΔNAB1
ΔLAB2 = ΔρAB2 + c·dtAB + λΔNAB2
站星双差 (Between-Satellite Double Difference)：
对两颗卫星的单差进行二次作差。可完全消除接收机钟差，且模糊度保持整数特性！
∇ΔLAB12 = ∇ΔρAB12 + λ∇ΔNAB12
